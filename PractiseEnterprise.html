About us

Welcome to our school project about StatBot! We are Lasse and Bart, studying ICT at Thomas More, and we have created a Discord bot that is designed to help users monitor their servers in real-time through a Grafana dashboard. StatBot is a unique tool that provides users with valuable data and analytics, including the number of users online, the number of messages sent, and the number of voice channels used. With this information, you can easily make informed decisions about your server and its users, and track your server's performance over time.

Discord bot

Our Discord bot, StatBot, is a powerful tool designed to help users monitor their servers in real-time. StatBot connects to your Discord server and gathers valuable data about user activity, channel usage, and more. It is incredibly user-friendly and easy to use. Simply add it to your server and start monitoring your server's performance. With StatBot, you can track metrics like the number of messages sent, the number of users online, and the number of voice channels used. This information can help you make informed decisions about your server and its users, and ensure that everyone is having a great experience.

Grafana dashboard

Our Grafana dashboard is an essential component of StatBot, providing users with real-time data and analytics about their Discord servers. With Grafana, you can easily monitor key metrics like user activity, channel usage, and more, and make informed decisions about your server and its users. The Grafana dashboard is designed to be user-friendly and easy to read, ensuring that you can quickly access the information you need. We believe that our Grafana dashboard is an essential tool for anyone who wants to take control of their Discord server and create a great online community.

SQlite database

Our SQLite database is a critical component of StatBot, facilitating communication between the Discord bot and Grafana dashboard. The database stores all the data collected by StatBot from your Discord server, allowing it to be easily accessed and analyzed on the Grafana dashboard. SQLite is a powerful and reliable database management system that is lightweight and easy to use. It ensures that StatBot can efficiently store and retrieve data, and enables real-time monitoring of key metrics like user activity, channel usage, and more.
